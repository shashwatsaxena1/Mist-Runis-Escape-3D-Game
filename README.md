# Rumours of the Mist: Escape — V1.1 Character Select

This is the updated V1 prototype with a character selection screen before gameplay.

## How to run
1. Extract the ZIP file.
2. Open the folder.
3. Double-click `index.html`.
4. Choose a character.
5. Click **Start Game**.

## Characters
- Yash: balanced starter character
- Vijaya: fast and brave
- Mikasa: low fear and longer torch
- Sam: high health and strong torch
- Raj: fastest but risky

## Controls
- Move: W A S D or Arrow Keys
- Torch: F
- Goal: collect 5 Natural Energy symbols, then reach the exit gate

## Files
- `index.html` contains the full game code.
- `assets/character_design.png` is used for the character selection card art.
