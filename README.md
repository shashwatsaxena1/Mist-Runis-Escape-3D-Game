# Rumours of the Mist: Escape — V1 Prototype

A beginner-friendly browser game prototype based on the manga world **Rumours of the Mist**.

## What is included

- One playable top-down horror level
- Yash as the playable character
- Living mist enemy that follows the player
- 5 Natural Energy symbols to collect
- Health, fear, and torch systems
- Locked exit that opens after all symbols are collected
- Game over and win screens
- Keyboard and simple mobile button controls
- No external assets or libraries required

## How to run

1. Open the folder.
2. Double-click `index.html`.
3. The game will open in your browser.

## Controls

- Move: `W A S D` or arrow keys
- Toggle torch: `F`
- Goal: collect all 5 symbols, then touch the exit gate
